The table view displays the current instance as a collection of tables. Table types include signatures, fields, and skolemized variables. Skolemized variables can optionally be viewed as highlighted rows in the signature or fields tables.

The table view only contains a single sidebar tool which provides general settings, as described in the following three sections.
