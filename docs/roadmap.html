General Features

  • Configurable Default Settings
  • Geometric Zooming — The Sterling graph view uses semantic zooming, in which the nodes of the graph are repositioned but not scaled in size, resulting in a sort of "spreading" action. This is useful when a graph is crowded and the user needs to focus in on a specific portion of the graph, but is less useful as the user zooms out. Geometric zooming, on the other hand, repositions and scales the nodes. This is less useful when zooming in, as the nodes simply get bigger and any sort of node crowding remains unchanged. For very large graphs, however, the ability to zoom out and scale nodes accordingly may provide additional context in, e.g., graphs that are too large to fit on the screen at a readable zoom level. It would be useful, then, to include a way to easily toggle between semantic and geometric zooming.
