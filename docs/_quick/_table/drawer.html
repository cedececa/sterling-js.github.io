The contents of the table drawer depend on the currently selected sidebar button. The following sections give a brief description of each drawer.

{% include h5-anchor.html text="Table Settings" %}

The Tables section allows you to choose the set of visible tables by category or by selecting individual tables. The skolems option will be available if the option to display skolems as highlighted rows is disabled. Click on the Choose Tables text field to display a list of tables to choose from, and begin typing to narrow the list using predictive search.

The Data Options section provides a number of toggles to adjust the visibility of certain tables and table attributes. With Display Skolems as highlighted rows active, skolem tables will not be visible and rows of the signature and field tables will be highlighted accordingly.

The Layout Options section provides buttons that can be used to adjust the layout and alignment of the tables on the stage.
